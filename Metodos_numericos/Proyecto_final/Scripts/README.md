## Compilación

Para ejecutar la compilación del programa se tiene que realizar los siguientes comandos en la carpeta principal.

```bash
make
```

## Ejecucción

Para ejecutar el archivo compilado de tiene que realizar el siguiente comando.

```bash
./main.out
```
