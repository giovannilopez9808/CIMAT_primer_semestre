\section{Ordenar información}

EL menú contiene descritos diferentes modos de ordenar la información de los estudiantes. Los criterios creados para el ordenamiento son los siguientes:

\begin{itemize}
    \item Ordenar por nombre
    \item Ordenar por edad
    \item Ordenar por promedio
\end{itemize}

Los criterios antes mencionados siguien el mismo algortimo de ordenamiento. EL algoritmo de ordenamiento planteado es el \script{quick\_sort}, el cual habia sido desarrollado en tareas anteriores. Esta función fue reimplementada para recibir como argumento una función el cual le indica que objeto (conjunto de carácteres o números) es mayor con respecto a otro.
